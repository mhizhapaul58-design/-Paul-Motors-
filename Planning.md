## **Design Principles**

### **Contrast**

Contrast is created by:

- Larger headings to show visual hierachy

- Using different font sizes for headings and body text

Helps users identify important sections

### **Repetition**

Repetition  is maintained by:

- Using the same layout structure across all pages 

- Repeating the same navigation on every page

This creates consistency and easier navigation

### **Alignment**

- Keeping text, images, and lists aligned vertically

This improves readability

### **Proximity**

- Placing images close to the text they relate to

This helps users understand which content belongs together
